# CYS
An open-source cybersecurity curriculum with team-based learning, labs, and certification mapping.

- **Tools/Platforms**: Kali Linux, Splunk, ELK, Wazuh, Zeek, Suricata, Sysmon, Autopsy, Volatility, Metasploit, Burp Suite, Aircrack-ng, BloodHound, TryHackMe, Hack The Box, Blue Team Labs Online, Security Onion.
- **Certifications**: Prepares for Security+, GCIA, OSCP, CRTP, CRTO, CHFI, GREM, CCNA Security.
- **Labs**: Emphasize hands-on practice with virtualized environments (VirtualBox, VMware) and CTF platforms.
- **Career Paths**: SOC Analyst, Blue Teamer, Penetration Tester, Cybersecurity Analyst.
